# Instagram-profile-ui

# Project Overview
This project is a front-end recreation of an Instagram Profile UI developed using HTML5 and CSS3. The objective of this project is to design and develop a clean, structured, and visually similar Instagram interface while following modern front-end development practices.

The project contains three different UI screens:
- Profile Screen
- Feed/Post Screen
- Post Grid Screen
The design focuses on maintaining proper spacing, alignment, typography, and organized layout structure.

## Features
- Instagram-inspired profile interface
- Profile information section with user statistics
- Story highlights layout
- Feed post design
- Post image gallery using grid layout
- Bottom navigation interface
- Clean and organized UI structure
- Separate image asset management
- Responsive layout design

## Technologies Used
- HTML5
- CSS3
- Flexbox
- CSS Grid
  
Flexbox is used for:
- Profile header alignment
- User statistics layout
- Navigation bar arrangement
- Story section alignment
- Horizontal content positioning

CSS Grid is used for:
- Post gallery layout
- Multi-column image arrangement
- Structured grid-based screen design

Semantic HTML elements are used to create a structured and readable layout, including:
- Header
- Section
- Navigation
- Image elements
- Content containers
